# anotações de fundamentos da progamação

## tipos de dados em python
1. string
2. number int
3. number float
4. boolean

## operadores matematicos - básicos
+ -> adição
- -> subtração
* -> multiplicação
/ -> divisão

## operadores logicos
and -> e -> se duas condições forem verdadeira, o resultado é verdadeiro
or-> e -> se pelo menos uma condição for verdadeira, o resultado é verdadeiro


## metodos em python
1. print() -> exibi informações no terminas
2. imput() -> capturar uma informação no terminal
## Format em python

 Estrutura de repetição
 `` if (se)`` -> Verifica se uma codição é true (verdadeire)
 ``elif (senão se)`` -> é usado para testar varias condições. Ele só executa se as condições anteriores forem falsas.
 ``else (senão)`` -> executa o código se a condição if for false.
